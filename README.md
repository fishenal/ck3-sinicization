## China History
综合各个朝代，采用接受度最高的名称
| English Title  |  Chinese Title |
| ------------- | ------------- |
| Baron | 县候 |
| Count | 县公 |
| Duke | 郡王 |
| King | 亲王 or 藩王 |
| Emperor | 皇帝 |
| Knight | 将军 |

采用和游戏历史时期大致相同的唐朝三省六部制

| Characters  |  Origin role | China title |
| ------------- | ------------- | ------------- |
| Spouse | 皇后，辅助 | 皇后 |
| Court Chaplain | 教皇 （中国并无此职位，试图改掉）| 唐朝三公级别高，虽无实权，可选取太师取代 |
| Chancellor | 外交官 | 礼部尚书   |
| Steward | 内务 |中书令, 宰相, 中书省的中书令，首席宰相 |
| Marshal | 军事 | 兵部尚书  尚书省兵部主管 或 （骠骑）大将军 |
| Spymaster | 间谍 | 刑部尚书 |
