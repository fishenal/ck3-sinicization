## China History
综合各个朝代，采用接受度最高的名称
| English Title  |  Chinese Title | English Version |
| ------------- | ------------- | ------------- |
| Baron | 县候 | Town Magistrate |
| Count | 县公 | County Magistrate|
| Duke | 郡王 | County King |
| King | 亲王 or 藩王 | Feudal King|
| Emperor | 皇帝 | Emperor/Empress |
| Knight | 将军 | General|
